## Project Overview

Xeetapus is a production-ready social media platform featuring:

- **Backend**: Custom Zig HTTP server with SQLite database
- **Frontend**: React 18 with TypeScript and Tailwind CSS
- **Security**: OWASP-compliant with comprehensive protection measures
- **Features**: Posts, comments, likes, follows, communities, DMs, polls, analytics, and more

### Tech Stack

|Component| Technology |
|-----------|------------|
|Backend Language | Zig 0.14.0+ |
|Database | SQLite3 |
|Frontend Framework | React 18.2 |
|Frontend Language | TypeScript 5.2 |
|CSS Framework | Tailwind CSS 3.4 |
|UI Components | Radix UI |
|Build Tool | Vite 5.0 |
|Task Runner | Just |
